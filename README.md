# Монитор договоров

Веб-приложение для отслеживания жизненного цикла договоров организации. Канбан-доски по разделам, база данных, отчёты, импорт Excel.

## Возможности

- **Канбан-доска** — договоры распределены по 5 разделам: Заключение → Исполнение → Изменение → Хранение → Архив
- **Перетаскивание** — drag-and-drop для сортировки внутри колонки
- **Переход между разделами** — при прохождении всех шагов раздела договор автоматически попадает во «Входящие» следующего
- **Дополнительные соглашения** — привязываются к основному договору, проходят через раздел «Изменение»
- **База договоров** — таблица с фильтрацией по разделу, типу, сумме, дате
- **Отчёты** — 6 предустановленных отчётов, сохранение пользовательских фильтров, экспорт в CSV
- **Импорт Excel** — drag-and-drop файла, автораспознавание колонок по русским названиям
- **Новости** — inline-редактор на главной странице

## Технологии

- Python 3.9+
- Flask + SQLAlchemy
- SQLite
- Bootstrap 5 + Bootstrap Icons
- Pandas (импорт Excel)

## Установка

```bash
# Клонировать репозиторий
git clone <url>
cd Dogovor

# Создать виртуальное окружение (рекомендуется)
python -m venv venv
venv\Scripts\activate  # Windows

# Установить зависимости
pip install flask flask-sqlalchemy pandas openpyxl
```

## Запуск

```bash
python app.py
```

Сервер будет доступен по адресу http://127.0.0.1:5000.

При первом запуске автоматически создаётся база данных `instance/dogovor.db` и загружаются 16 образцовых договоров + 6 новостей.

## API endpoints

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Главная страница |
| GET | `/board` | Доска разделов |
| GET | `/board/<section>` | Канбан раздела |
| GET | `/database` | База договоров |
| GET | `/reports` | Отчёты |
| GET | `/api/contracts` | Список договоров (с фильтрами) |
| GET | `/api/contracts/<section>` | Договоры раздела |
| POST | `/api/contract/<id>/move` | Переместить по шагам |
| POST | `/api/reorder` | Изменить порядок в колонке |
| PUT | `/api/contract/<id>` | Обновить договор |
| DELETE | `/api/contract/<id>` | Удалить договор |
| POST | `/api/contract/<id>/additional` | Создать ДС |
| POST | `/api/import` | Импорт Excel |
| GET | `/api/stats` | Статистика |
| GET/POST/DELETE | `/api/news` | Новости CRUD |
| GET/POST/DELETE | `/api/reports` | Отчёты CRUD |
| POST | `/api/shutdown` | Остановить сервер |
